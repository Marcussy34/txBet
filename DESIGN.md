# txBet interface system

The interface is a dense broadcast-market console, not a generic SaaS dashboard.

- Dark-first Split Signal palette from [`brand.md`](brand.md).
- Sharp borders and compact spacing communicate execution discipline.
- Barlow Condensed owns headlines; IBM Plex Mono owns prices, clocks, state, and controls.
- Custom product glyphs represent agents, feed state, matching, blocks, and leg risk.
- Lucide icons are allowed only for standard utility controls.
- Motion is reserved for tape transitions and candidate emergence; reduced-motion preferences are respected.
- Mobile layouts stack panels and preserve every disclosure and risk state without horizontal page overflow.

The pitch disclosure must remain visible: synthetic TxLINE-format replay, simulated venue books, simulated execution.
